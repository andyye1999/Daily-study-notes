
# 最大公约数

```cpp
int gcd(int a, int b)
{
    while (b != 0)
    {
        int tmp = a;
        a = b;
        b = tmp % b;
    }
    return a;
}
```

或者更简洁

```cpp
int gcd(int a,int b)
{
    return b?gcd(b,a%b):a;
}
```


# 判断一个数是不是质数

```cpp
bool is_prime(int x) {
    if (x <= 1) {
        return false;
    }
    for (int i = 2; i <= x / i; i++) {
        if (x % i == 0) {
            return false;
        }
    }
    return true;
}
```

# 二进制中1的个数

巧用 n&(n−1)  
把一个整数减去1，再和原整数做与运算，会把该整数最右边一个1变成0.那么一个整数的二进制有多少个1，就可以进行多少次这样的操作。
```cpp
class Solution {
public:
    int hammingWeight(uint32_t n) {
        int res = 0;
        while(n)
        {
            n = n & (n-1);
            res++;
        }
        return res;

    }
};
```
